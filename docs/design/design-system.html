Design System · v1.0

RUNECHAIN

Cozy-gothic Soulslike with bureaucratic horror aesthetic. Monospace-first, pixel-perfect, dark panels. Gold for economy, rune-green for power, blood-red for danger. All tokens are CSS custom properties on :root. No build step required — all styling lives in index.html.

--mana
#7aa7ff
Mana, chat. Player names in chat, chat input border, mana bar.
--txt
#f1ead8
Primary text. Warm off-white — avoids cold white on dark panels.
--muted
#a9a08c
Labels, secondary text, metadata. Used with .lab class.

All tokens are declared on :root in the <style> block of index.html. Structural colours not in :root — such as #050607 (shadow/outline), #191b1d (bar trough), #191d20 (button bg) — are used literally and intentionally kept unlabelled to avoid over-abstraction.
